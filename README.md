# 🌿 韭菜MBTI人格分析

> 16道投资场景趣味题，测测你是哪种韭菜！

将 MBTI 人格测试与 A 股韭菜文化结合的微信小程序。纯前端实现，无需后端服务，2 分钟出结果，16 种韭菜人格等你认领。

## 截图

| 首页 | 答题页 | 结果页 |
|:---:|:---:|:---:|
| 🌱 韭菜绿渐变背景 + 浮动叶片动画 | 📊 进度条 + 滑动切题动画 | 🎯 韭菜人格解读 + 维度分析 |

## 功能特色

- **16 道投资场景题** — 从社交、决策、逻辑、纪律四个维度测试你的韭菜气质
- **16 种韭菜人格** — 每种含专属 emoji、人格解读、标签、投资忠告、优劣势分析
- **4 维度雷达** — 社交型 vs 独狼、技术流 vs 玄学派、理性派 vs 感性派、计划型 vs 随性型
- **一键分享** — 分享给股友一起测，看看谁是同一个韭菜品种
- **纯前端计算** — 无后端，无需注册，数据不上传

## 技术栈

- **框架**：微信小程序原生开发（WXML / WXSS / JS）
- **数据流**：`app.globalData` 全局状态（scores / mbtiType / answers）
- **计算**：纯前端 MBTI 维度得分计算，无服务端依赖
- **基础库**：3.16.0+

## 快速开始

### 前置条件

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 微信小程序 AppID（已在 `project.config.json` 中配置）

### 运行

1. 克隆项目：
   ```bash
   git clone https://github.com/SailFree-Elmo/JiuCai-MBTI-Test.git
   ```
2. 用微信开发者工具打开项目目录
3. 编译运行即可预览

### 配置

- AppID 在 `project.config.json` 的 `appid` 字段修改
- 小程序功能介绍等文案见 `docs/mini-program-intro.md`

## 页面导航流

```
首页(index) ──navigateTo──> 答题页(question) ──redirectTo──> 结果页(result)
    ↑                                                    │
    └──────────────────── reLaunch ───────────────────────┘
```

- 首页 → 答题页：`wx.navigateTo`（可返回）
- 答题页 → 结果页：`wx.redirectTo`（关闭答题页，防回退改答案）
- 结果页 → 重新测试：`wx.reLaunch`（清空页面栈回首页，防栈堆积）
